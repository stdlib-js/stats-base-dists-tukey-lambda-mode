
{{alias}}( λ )
    Returns the mode of a Tukey lambda distribution with shape parameter `λ`.

    If provided `NaN`, the function returns `NaN`.

    Parameters
    ----------
    λ: number
        Shape parameter.

    Returns
    -------
    out: number
        Mode.

    Examples
    --------
    > var y = {{alias}}( 1.0 )
    0.0
    > y = {{alias}}( 2.0 )
    0.0
    > y = {{alias}}( NaN )
    NaN
    > y = {{alias}}( 0.0 )
    0.0
